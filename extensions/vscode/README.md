## What You Can Do

- Chat with your codebase from the Wasp Code sidebar
- Edit code with natural-language instructions
- Use inline autocomplete and next-edit suggestions
- Configure your own providers and models, including local and hosted models
- Reuse existing `.continue` config compatibility paths where supported

## Getting Started

1. Install the extension.
2. Open the **Wasp Code** sidebar in VS Code.
3. Add or select your models from the settings/config flow.
4. Start a new session and chat with your project.

## Configuration

Wasp Code supports the built-in settings UI as well as compatible config files used by the extension.

Typical config locations include:

- `~/.continue/config.yaml`
- `~/.continue/config.json`

## Features

### Chat

Ask questions about your code, request changes, and work through development tasks in-context.

### Edit

Select code and apply targeted edits without leaving your file.

### Autocomplete

Get inline suggestions while typing and use next-edit flows for faster iteration.

### Sessions and History

Keep past conversations available from the sidebar and resume work quickly.

### Model Flexibility

Use hosted providers, local models, or custom API-compatible setups depending on your workflow.
