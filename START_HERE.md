## 🚀 Quick Start (Right Now)

### Step 1: Install

```bash
npm install
```

### Step 2: Verify

```bash
npm run tsc:check
```

### Step 3: Start Development

Open 3 terminals:

**Terminal 1:**

```bash
npm run tsc:watch
```

**Terminal 2:**

```bash
cd gui && npm run dev
```

**Terminal 3:**

```bash
cd extensions/vscode && npm run esbuild-watch
```

### Step 4: Test

- Open VS Code
- Press `F5` to launch the extension
- Press `Ctrl+L` to open Wasp Code chat

- **Language:** TypeScript

- **Backend:** Express.js, Socket.io
