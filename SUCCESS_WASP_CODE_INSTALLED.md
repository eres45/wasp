## 🚀 How to Use

### 1. Open VS Code

VS Code should have opened automatically. If not, run:

```bash
code .
```

### 2. Find Wasp Code

- Look for the **"Wasp Code"** icon in the Activity Bar (left sidebar)
- It should appear as a new icon alongside Explorer, Search, etc.

### 3. Open the Chat Panel

- Click the Wasp Code icon to open the sidebar
- You should see the chat interface load

### 4. Test Keyboard Shortcuts

- **Ctrl+L**: Focus chat input (clears chat and adds selected code)
- **Ctrl+I**: Edit mode for selected code
- **Ctrl+Shift+L**: Add to chat without clearing

### 5. Verify MiniMax Models

The extension should automatically detect your MiniMax models from `~/.continue/config.json`:

- MiniMax-M2.1
- MiniMax-M2.5
- MiniMax-M2.7

## 🔧 Troubleshooting

If you don't see the Wasp Code icon:

1. **Check Extensions View**:

   - Go to Extensions (Ctrl+Shift+X)
   - Search for "Wasp Code"
   - Make sure it's enabled

2. **Check Developer Console**:

   - Press F12 to open Developer Tools
   - Look at Console tab for any errors
   - Extension should load without errors now

3. **Restart VS Code**:
   - Close all VS Code windows
   - Reopen VS Code
   - Extension should appear
