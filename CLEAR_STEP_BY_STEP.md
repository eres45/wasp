# 🎯 VERY CLEAR STEP-BY-STEP INSTRUCTIONS

## Step 1: Look at the LEFT SIDEBAR

1. Look at the very LEFT side of VS Code
2. You'll see a vertical bar with icons
3. Find the icon that looks like a "play button with a bug" (🐛▶️)
4. **CLICK on that icon** - this opens "Run and Debug"

## Step 2: Look at the Run and Debug Panel

After clicking the debug icon, you'll see a panel that says "RUN AND DEBUG" at the top.

In this panel, look for:

- A dropdown box at the top
- It might say "No Configurations" or show some options
- **CLICK on this dropdown box**

## Step 3: Select "Run Extension"

In the dropdown, you should now see:

- "Run Extension" (this is what we want!)
- Maybe some other options

**CLICK on "Run Extension"**

## Step 4: Click the Green Play Button

After selecting "Run Extension":

- You'll see a green triangle/play button (▶️)
- It's usually right next to the dropdown
- **CLICK the green play button**

## Step 5: Wait for New Window

VS Code will:

- Show some building/compiling messages
- Open a NEW VS Code window
- The new window title will say "Extension Development Host"

## Step 6: Test in the New Window

In the NEW window (Extension Development Host):

- **Press Ctrl + L**
- **Look for Wasp Code icon in the left sidebar**
- **Try typing in the search: Ctrl + Shift + P, then type "wasp"**

1. **Close VS Code completely**
